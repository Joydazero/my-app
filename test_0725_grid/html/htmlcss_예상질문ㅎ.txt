🎯 HTML & CSS 인터뷰 연습지

1. ✅ 시맨틱 태그란 무엇인가요?
💬 면접관 질문:
시맨틱 태그가 무엇이며 왜 사용하는지 설명해 주세요.
🧠 답변 요지:
* 의미가 담긴 HTML 태그
* 구조 파악 쉬움, SEO와 접근성 향상

(직접 작성해보세요)

2. ✅ HTML 기본 문서 구조를 설명해보세요.
💬 면접관 질문:
HTML의 기본 틀을 어떻게 구성하나요?
🧠 답변 요지:
* <!DOCTYPE html>, <html>, <head>, <body>
* 메타 정보와 본문 콘텐츠 구분


3. ✅ 클래스와 아이디의 차이는?
💬 면접관 질문:
클래스와 아이디는 어떤 점이 다른가요?
🧠 답변 요지:
* . 클래스: 여러 요소에 적용 가능
* # 아이디: 유일한 값, 1개 요소에만



4. ✅ 박스 모델에 대해 설명해보세요.
💬 면접관 질문:
CSS 박스 모델이 무엇인가요?
🧠 답변 요지:
* content, padding, border, margin
* 요소의 실제 크기 계산과 레이아웃에 중요



5. ✅ position 속성의 종류와 차이점은?
💬 면접관 질문:
CSS position 속성에 대해 설명해주세요.
🧠 답변 요지:
* static, relative, absolute, fixed, sticky
* 위치 기준 및 흐름에서의 영향이 다름



6. ✅ Flexbox의 주축과 교차축은 무엇인가요?
💬 면접관 질문:
Flexbox에서 justify-content와 align-items의 기준은?
🧠 답변 요지:
* justify-content: 주축 기준 정렬
* align-items: 교차축 기준 정렬
* 주축은 flex-direction에 따라 바뀜


7. ✅ rem과 em의 차이는 무엇인가요?
💬 면접관 질문:
rem과 em은 모두 상대 단위인데, 차이가 뭔가요?
🧠 답변 요지:
* em: 부모 요소의 폰트 크기 기준
* rem: html 기준 (루트 기준)
* 중첩되면 em은 누적, rem은 일정


8. ✅ display: block과 inline의 차이는?
💬 면접관 질문:
block 요소와 inline 요소는 어떻게 다르죠?
🧠 답변 요지:
* block: 한 줄 전체 차지, div, p 등
* inline: 콘텐츠 크기만큼 차지, span, a 등



9. ✅ CSS에서 선택자 우선순위는 어떻게 계산하나요?
💬 면접관 질문:
어떤 스타일이 적용될지 결정하는 우선순위 기준은요?
🧠 답변 요지:
* 인라인 > 아이디 > 클래스 > 태그
* 같은 수준이면 나중에 선언된 것이 우선


10. ✅ 미디어쿼리란 무엇인가요?
💬 면접관 질문:
반응형 웹을 구현할 때 미디어쿼리는 어떻게 활용하나요?
🧠 답변 요지:
* 화면 너비/장치 조건에 따라 스타일 변경
* @media (max-width: 768px) 등 사용




11. ✅ alt 속성의 역할은 무엇인가요?
💬 면접관 질문:
<img> 태그의 alt 속성은 어떤 역할을 하나요?
🧠 핵심 포인트:
* 이미지가 보이지 않을 때 대체 텍스트 제공
* 시각 장애인을 위한 스크린 리더용 설명
* 접근성(Accessibility) 향상


12. ✅ <label> 태그의 사용 목적은?
💬 면접관 질문:
<label> 태그를 왜 쓰나요? for 속성은 무슨 역할인가요?
🧠 핵심 포인트:
* <input>과 연결해 클릭 가능 범위 확장
* for는 input의 id와 연결
* UX 및 접근성 향상



13. ✅ z-index는 어떻게 작동하나요?
💬 면접관 질문:
z-index 속성은 어떤 역할을 하나요?
🧠 핵심 포인트:
* 요소의 쌓이는 순서 조정
* position이 있어야 작동 (relative, absolute 등)
* 값이 클수록 위로 올라감



14. ✅ visibility: hidden과 display: none의 차이는?
💬 면접관 질문:
두 속성의 차이를 설명해보세요.
🧠 핵심 포인트:
* display: none: 공간까지 사라짐
* visibility: hidden: 공간은 남고 보이지 않음



15. ✅ min-width, max-width는 언제 사용하나요?
💬 면접관 질문:
min-width와 max-width의 사용 예를 들어주세요.
🧠 핵심 포인트:
* 콘텐츠가 너무 작거나 커지지 않도록 제한
* 반응형 디자인에서 자주 사용됨
* min-height, max-height도 동일 원리


16. ✅ float과 clearfix는 무엇인가요?
💬 면접관 질문:
float로 정렬했는데 부모 높이가 사라졌어요. 왜죠?
🧠 핵심 포인트:
* float은 요소를 문서 흐름에서 분리
* 부모가 자식 높이를 인식하지 못함
* clearfix로 해결 (overflow: hidden 또는 가상요소 활용)



17. ✅ Grid와 Flexbox 차이는?
💬 면접관 질문:
Grid 레이아웃과 Flexbox의 차이를 설명해주세요.
🧠 핵심 포인트:
* Flex: 1차원 레이아웃 (가로 or 세로 정렬)
* Grid: 2차원 레이아웃 (행 + 열)
* 복잡한 격자형 구조는 Grid가 더 유리



18. ✅ 반응형 웹을 만들기 위한 방법은?
💬 면접관 질문:
반응형 웹 디자인은 어떻게 구현하나요?
🧠 핵심 포인트:
* media query, %, vw/vh, rem
* 유동적인 단위와 조건부 스타일링 활용
* 모바일/태블릿 대응



19. ✅ 웹 접근성이란?
💬 면접관 질문:
웹 접근성이란 무엇이고 왜 중요한가요?
🧠 핵심 포인트:
* 장애 유무와 관계없이 모두가 사용 가능한 웹
* 스크린 리더, 키보드 접근 등 고려
* 시맨틱 태그, alt, aria-* 속성 등 사용


20. ✅ CSS 변수(CSS Custom Property)란?
💬 면접관 질문:
CSS 변수는 어떻게 사용하나요?
🧠 핵심 포인트:
* --변수명 선언, var(--변수명) 호출
* 유지보수 편리, 테마 설정 유용
* 예:
:root {
  --main-color: #333;
}
h1 {
  color: var(--main-color);
}
